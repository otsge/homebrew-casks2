# OTSGE Casks (2)

## How do I install these formulae?

`brew install otsge/casks2/<formula>`

Or `brew tap otsge/casks2` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "otsge/casks2"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
